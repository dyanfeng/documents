# 条件
## 条件结构
1. If条件结构  
  - 根据条件判断之后再做处理  

  ```
  1. 基本类型
  if (条件) {
    //满足条件的语句
  }  

  2. if... else...
  if (条件) {
    //满足条件的语句1
  }else {
    //不满足条件的语句2
  }

  3. 多重if 结构
  if (条件1) {
    //满足1条件的语句
  }else if(条件2) {
    //不满足1条件,判断是否满足2,执行满足2的语句
  }else if(条件3) {
    //满足条件3的语句
  }else{
    //以上条件都不满足的语句
  }
  ```  
2. switch 条件结构  

```
switch (表达式) {
  case 常量1:
    语句;
    break;
  case 常量2:
    语句;
    break;
  default:
    语句;//以上条件都不满足,执行默认语句
}
注意:1.表达式可以是byte,short , char,int ,enum,或者是string
    2.break 可以省略掉,但会一直执行,直到出现break ,否则,会执行到结束,称为穿透
    3. default 可以省略,也可以不出现在最后
    4. long 类型, float ,double,不可以直接匹配,要转换数据类型
    5. boolean 类型,不可以与其他类型进行比较

```    

3. If  和   switch 的比较    

```
相同点: 都可以实现多分支结构
不同点: 1. switch ,只能处理等值的条件判断,且条件是整型变量,或字符变量     的等值判断  
       2. 多重if :处理在else部分还包含其他if 结构,特别适合某个变量处于某个区间时候的情况  
注意:switch 可以很容易转成if ,而If很难转换成switch

```
